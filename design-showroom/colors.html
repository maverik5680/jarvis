Theming

We personalize services to offer every customer the best possible user experience tailored to their needs.

Brand Communication

The brand is front and center. Jarvis manages theme usage. Light theme is primary, while dark elements create contrast and emotion.

Function

Focus is on functionality (e.g., apps). The user can switch between light and dark themes based on preference. Light theme is default.

Elevations

Elevations give the interface a sense of three-dimensionality and depth. They establish clear hierarchies and help the user find their way around.
