Page Layouts

Standard page structure.

<body class="page">
  <header class="page-header">
    <div >Header</div>
  </header>
  
  <main class="page-content">
    <div >
       Content goes here...
    </div>
  </main>

  <footer class="page-footer">
    <div >Footer</div>
  </footer>
</body>

The .container class provides responsive max-width constraints.
