Buttons

Primary Buttons

The filled-out primary button is used for the most important action on a page.

<button class="btn btn--primary-black">Primary Black</button>
<button class="btn btn--primary-white">Primary White</button>

Secondary Buttons

The transparent secondary button with a thin frame is used for subordinate actions.

<button class="btn btn--secondary-black">Secondary Black</button>
<button class="btn btn--secondary-white">Secondary White</button>

Text Buttons

Subtle call to action, typically combined with an arrow.

Size Specification

Buttons are scaled based on font size with a minimum width of 9x the base font size.

Small (Base 16px, Min-width 144px)
Medium (Base 18px, Min-width 162px)
Large (Base 20px, Min-width 180px) 
<!-- Small: 16px / 144px min-width -->
<button class="btn btn--sm">...</button>

<!-- Medium: 18px / 162px min-width -->
<button class="btn btn--md">...</button>

<!-- Large: 20px / 180px min-width -->
<button class="btn btn--lg">...</button>

System Icon Buttons

Transparent buttons for classic interactions (save, print, download).

Button Groups

Grouped actions with a specification gap of 0.25rem (4px).

Primary & Secondary Group

Secondary Group

<div class="btn-group">
  <button class="btn btn--primary-black">Action 1</button>
  <button class="btn btn--secondary-black">Action 2</button>
</div>
