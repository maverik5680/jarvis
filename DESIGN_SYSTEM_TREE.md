# Jarvis Design System Tree

This document provides a complete overview of the Jarvis repository structure.

## Root Directory Structure

```
jarvis/
├── .gitignore
├── .vscode/
├── README.md
├── ds/
├── feedback/
├── index.html
├── js/
├── node_modules/
├── package-lock.json
├── package.json
├── templates/
├── vite-config/
├── vite.config.js
└── vscode-logs.log
```

## Directory Details

### `.vscode/`
IDE configuration directory for Visual Studio Code settings

### `ds/`
Design system components and assets directory

### `feedback/`
Feedback-related components or documentation

### `js/`
JavaScript source files

### `node_modules/`
Node.js dependencies (generated by npm/yarn)

### `templates/`
Template files for the design system

### `vite-config/`
Vite bundler configuration files

## Configuration Files

- **`.gitignore`** - Git ignore patterns
- **`package.json`** - Node.js project configuration and dependencies
- **`package-lock.json`** - Locked versions of dependencies
- **`vite.config.js`** - Vite build tool configuration

## Main Files

- **`index.html`** - Main HTML entry point (1.5 KB)
- **`README.md`** - Project documentation
- **`vscode-logs.log`** - VS Code logging file (23.6 KB)
